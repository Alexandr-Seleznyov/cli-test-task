# Parser
## Installation
```bash
composer require "cli-test-task/parser @dev"
```
Или [скачать](https://github.com/Alexandr-Seleznyov/cli-test-task)

## Usage
Перейти в папку
```bash
parser\src
```
Команды:

```
parse [url]
report [domain]
help
```
Если команды не отрабатывают, тогда:
```
php parse.php [url]
php report.php [domain]
php help.php
```
Или измените
```
#!/usr/bin/php
```
